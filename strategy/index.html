現状認識 ― なぜ「回し方」を変えるのか

精神論や努力不足ではない。仕組みが生んだ3つの構造要因。

1

一人で回す設計だった

4 Value Cycle は徳永一人が全工程を循環させる前提。一人分の動きをそのまま全員に求めた時点で、移植できない設計だった。

2

CK月次が中長期を殺す

評価は半期だがCKは毎月追う。成果に時間のかかるアカウント開拓は後回しに。達成意識が高い人ほど目先へ向かう。

3

FS起点の施策は形骸化

工数を増やす施策は防衛本能で淘汰される。送付除外、ボタン放置。FSの能動的アクションを入り口にした施策は必ず死ぬ。

結論： 変えるのは「何をやるか（What）」ではない。「どう回すか（How / Who）」である。

この戦略の文脈 ― なぜ今、戦い方を変えるのか

CONTEXT ① ｜ THE MODEL型からの脱却

パイプラインを「進めるだけ」では、もう勝てない。
役割が重なり合う時代の、一貫性の戦い。

これまでは、ISから振られたパイプラインを進捗させることだけにフォーカスしてきた。だが今後はそれだけでは難しい。これは当社のIS人員の問題ではなく、世の中のGTM戦略そのものが変化している ── 各役割がオーバーラップし、一人では完結できない。だからこそ役割分担して注力する。しかし分けるほど、情報・打ち出し・施策・コミュニケーションに「一貫性」が必要になる。

従来：THE MODEL型

IS→FS→CS が分業・分断。各自が自工程だけを回す。

これから：一貫性型

マーケ／IS／FS／CS が重なり合う。全レイヤーで一貫して動く。

空中戦：マーケ 地上戦：ISコール 地上戦：FS提案 地下戦：CS活動

全レイヤーの活動に一貫性を持たせる ── だから「戦略」が要る。

CONTEXT ② ｜ 解決策ベンダーへ

「プロ人材」という手段売りと、
「解決策」の両方を取りに行く。

「顧客思考」はマインドとして当然。その上で、動きとして「解決策ベンダー」の力を持つ必要がある。「プロ人材」という手段売り（FLEXY）はAIマッチングの加速で人の稼働確保を強化し、「解決策」の提案（プロシェア案件）は提案力の強化で案件の領域を広げる。テーマの切り口を切る・体制を提案する・使い方/アサイン/役割を提示する ── JD（人材要件）自体をこちらが描くイメージ。手段売りと解決策の両方を取りに行くことが、競合に勝つ条件になる。

「人材・プロ人材」というソリューションが決まっていれば、当社は「人材」を提供すればよかった。だが今、企業は様々なサービスで「解決」を考えられる。だから人ではなく、「解決策」を一緒に考え検討するフェーズから顧客と対話する必要がある。

FSの役割を2層に分ける ― 保有案件数と強みに応じた配置

全員が同じ動きをするのではなく、強みに応じて役割を分ける

ソリューション営業への変革は全員一律ではできない。保有案件数と実証された強みに応じて、FSの動きを Farming（深耕強化） と Hunting（開拓強化） の2つに分け、役割を純化する。

Farming（深耕強化）

継続の深耕 ＋ 既顧客新規

保有Tier1中心・保有案件が厚い層
継続を守る。新規開拓ではなく、既存の深いリレーションを活かした継続受注・追加ロール開拓に集中する。更新は6ヶ月・1年スパンの提案へ。
既顧客から追加案件を獲得する。温まった関係を起点に、同一顧客の新規案件を取りに行く（既顧客新規のロールモデル）。
“守り”が主、“攻め”は従。活動比重は Tier1の進行営業（継続・深耕）8割 : 新規開拓 2割 を目安に（チーム方針であり、個人への指示ではない）。
Hunting（開拓強化）

新規パイプラインの構築

Tier2以下が軸・保有案件が積み上げ途上の層
新しいパイプラインを作る。Tier2以下を軸に、マーケ連携・NDA巻き・Tier引き上げで新規案件を生む。ソリューション提案への変革の主軸。
CS活動も薄く担い、稼働案件のフォローを行う。
案件数を積み上げながら、提案力・深耕の幅を広げていく。
Farming と Hunting は「0 vs 100」ではない。
どちらの層も新規も深耕も行う。変えるのは 比重＝どこに向けるか であって、活動の「量」ではない。
Farming｜深耕強化

狙い

Tier1・稼働中顧客という「資産」の関係の深さを活かし、既存アカウントからの追加ロール・新規部署への展開を掘り続けることに集中する。

期待値

新規を断るという意味ではない。文脈が薄いところからゼロで文脈構築する動きは減らし、既に文脈があるところをより濃くしていく比重に寄せる、という意味。活動の「量」を減らすのでなく、「どこに向けるか」を変える。

Hunting｜開拓強化

狙い

関係値がまだ薄い顧客・新規開拓を主戦場とし、新しいパイプラインを作ることに集中する。

期待値

Farming側が長年かけて積み上げたTier1資産を、ゼロから同じように作るのは現実的でない。新規開拓を「頑張れ」と精神論で終わらせず、マーケティングを強化してHunting側への新規リード供給を増やす。開拓は個人の営業力だけに頼らせない。

残された論点 ― まだ解けていないこと

正直に認識し、向き合う。隠さない。

①

新規アカウントの「突破力」は型化できるか

今期の施策の多くは既に関係のある顧客が起点。だが「そもそも会ってもらう・返信が来る関係をゼロから作る力」は依然属人的で、型化の目処が立っていない。外部知見も得ながら探求する。

②

評価をCKと切り離せるか

中長期の関係構築を促すには、CK一辺倒から活動量（到達人数・定例同席・インバウンド発生）評価への移行が要るかもしれない。戦略アカウントを「報酬」でなく「責任」と定義できるか。
